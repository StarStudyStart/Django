４、mysql远程用户访问

  • 第一步 授权root用户远程登录权限

    mysql> USE mysql;
    mysql> SELECT User, authentication_string, host form user;
    mysql> GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' IDENTIFIED BY 'pwd''
    mysql> FLUSH PRIVILEGES
    权限：表示 ‘select’、‘update’、‘drop’等权限
    *.* ：表示 '*'数据库.'*'表名
    MYSQL 把所有权限都放进了cached中，所以更改后要重新加载

  • 第二步 开放防火墙端口

    firewall --cmd

5、mysql -u root -p的含义

> -h 指出服务器名 localhost是指本机 -u 指出用户名 root是默认的最高权限用户名 -p指出密码 不填就指密码为空

6、root 用户操作

新建用户修改权限

  1. 查看用户

    mysql> select user, host, authentication_string from mysql.user;

  2. 新建用户

    mysql> create user 'usename'@'host' IDENTIFIED BY 'password'

  3. 授权用户

    mysql> grant all privileges ON *.* TO 'username'@'host' IDENTIFIED BY 'password'

  4. 删除用户

    mysql> DROP user 'username'@'host';

更改密码

  1. 需要登录mysql，适用于初次安装更改密码以及知道密码的情况

    mysql> set password for 用户名@localhost = password('新密码');

  2. 在mysql系统外，使用mysqladmin更改密码

    mysqladmin -u root -p password "test123"
    Enter password: 【输入原来的密码】

  3. 使用skip跳过密码登录，登录mysql

    mysqld_safe --skip-grant-tables &&
    mysql
    mysql> use mysql;
    mysql> update user set password=password("123456") where user='root';
